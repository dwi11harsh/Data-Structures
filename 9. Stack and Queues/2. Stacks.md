## Implementation

- implementation of stack (unlike other data structures) takes same time with linked list as well as array
### Using Array
- there are only two ops in stacks
	- push
	- pop
```c
int stack [MAX]
int top = -1;

void push ( int item )
{
	if (top==MAX-1)
	{
		printf("overflow");
		return;
	} else {
		stack[++top] = item;
	}
}
```
